## 国际化

MDx 支持多语言，默认语言为简体中文。

支持的语言如下：

- 简体中文
- 土耳其语（感谢 [Hasan CAN](https://github.com/Sn0bzy)）

计划支持的语言如下：

- 英语

> 非常欢迎你帮助我们将 MDx 翻译至其他语言！
